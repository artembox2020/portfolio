Развертка и настройка CRM-системы

1)Склонировать проект с репозитория

2)Файлы с суффиксом -sample скопировать в файл с тем же названием (но без суффикса) и модифицировать согласно вашим локальным настройкам

3)Создать базу, конфиг прописать в ConfigLoaderBase.php, дамп взять с q77626bj_crm.sql

После этого проект должен запуститься, если в процессе развертки возникают проблемы, пишите мне на почту artem.palamarchuk@webpard.de

Для входа под админом используйте логин/пароль admin/admin либо demo/demo
